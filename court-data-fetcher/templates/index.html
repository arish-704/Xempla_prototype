How to Use This System

1

Fill Case Details

Select the case type, enter the case number and year in the form above

2

Submit Search

Click "Fetch Case Data" button to initiate the search process

3

Browser Window Opens

A new browser window will open showing the court website

4

Solve CAPTCHA

Important: You must solve the CAPTCHA and click "Search" button manually

5

View Results

The system will automatically fetch and display the case information

This system requires manual CAPTCHA solving to comply with court website security measures. The process ensures authentic access to public court records.
